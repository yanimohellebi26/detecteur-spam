# AI Spam Detector with Telegram Bot

## Tagline
ML spam classifier with TF-IDF + Logistic Regression deployed as a live Telegram bot.

## Description
A complete machine learning pipeline for spam detection — from raw text preprocessing (stemming, stopword removal) through TF-IDF vectorization and logistic regression training, to real-time inference via a live Telegram bot. Includes model evaluation with confusion matrix and ROC curves, and adjustable confidence thresholds.

## Motivation
Demonstrating the full ML lifecycle: data preprocessing → feature engineering → model training → evaluation → real-time deployment via a messaging platform.

## Tech Stack
- **Language**: Python
- **ML**: scikit-learn, TF-IDF, Logistic Regression, NLTK
- **Deployment**: Telegram Bot API
- **Data**: Pandas, Matplotlib
- **Serialization**: joblib

## Key Features
- TF-IDF vectorization + logistic regression classifier
- Adjustable confidence threshold for spam detection sensitivity
- Real-time Telegram bot with contextual responses
- Model evaluation with confusion matrix and ROC curve
- Text preprocessing pipeline: stemming, stopword removal, tokenization

## Skills Demonstrated
- End-to-end ML pipeline development
- NLP text preprocessing and feature engineering
- Model evaluation and metrics analysis
- Bot development and API integration
- Real-time ML inference deployment
